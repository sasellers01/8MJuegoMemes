Clase SistemaMemes

Clase principal del juego Memes 8M. Contiene todos los métodos estáticos que implementan las Historias de Usuario (HU1–HU10) y las clases internas de datos Realidad y Puntuacion.

Detalle de métodos

static boolean comprobarFicherosDatos()

HU1 — Comprueba que existen los tres ficheros de datos obligatorios: datos/memes.txt, datos/realidades.json y datos/soluciones.xml. Si alguno no existe imprime un mensaje de error.

Devuelve:boolean true si todos existen, false en caso contrario
static void comprobarFicheroResultados()

HU2 — Comprueba que existe resultados/mejores.txt. Si no existe crea el directorio y el fichero vacío.

Lanza:IOException si hay un error al crear el fichero
static Map<Integer, String> leerMemes(String ruta)

HU3 — Lee el fichero de memes línea a línea. Ignora líneas vacías y comentarios (que empiezan por #). Separa cada línea por el delimitador | con límite 2 para no romper textos que contengan el carácter pipe.

rutaRuta relativa al fichero datos/memes.txt
Devuelve:Map LinkedHashMap<Integer, String> con id como clave y texto como valor, en orden de inserción
Lanza:IOException si el fichero no se puede leer
static List<Realidad> leerRealidades(String ruta)

HU4 — Parsea el fichero JSON manualmente sin librerías externas. Divide el contenido por el patrón {"id" para aislar cada entrada. Usa los helpers extraerEntero(), extraerJsonAnidado() y extraerJson().

rutaRuta relativa al fichero datos/realidades.json
Devuelve:List ArrayList<Realidad> con todas las realidades cargadas
Lanza:IOException si el fichero no se puede leer
static int mostrarMemeYRealidades(Map<Integer,String> memes, List<Realidad> realidades, List<Integer> memesUsados, int puntuacion)

HU5 — Selecciona aleatoriamente un meme de los disponibles (no usados), busca su realidad correspondiente y muestra las dos opciones mezcladas. Añade el memeId seleccionado a memesUsados para evitar repeticiones.

memesMapa con todos los memes cargados
realidadesLista con todas las realidades cargadas
memesUsadosLista de IDs ya mostrados (se actualiza en este método)
puntuacionPuntuación actual para mostrar en el marcador
Devuelve:int Número de opción correcta (1 o 2) para que HU6 lo compruebe
static int comprobarRespuesta(Scanner scanner, int opcionCorrecta, int puntuacion)

HU6 — Lee la respuesta del usuario y la valida (debe ser 1 o 2). Si coincide con la opción correcta suma un punto y muestra un mensaje positivo. Si es incorrecta muestra un mensaje de error sin modificar la puntuación.

scannerEscáner de entrada estándar
opcionCorrectaNúmero (1 o 2) de la opción correcta devuelto por HU5
puntuacionPuntuación antes de esta ronda
Devuelve:int Nueva puntuación tras evaluar la respuesta
static int jugar(Map<Integer,String> memes, List<Realidad> realidades, Scanner scanner)

HU7 — Ejecuta el bucle principal del juego durante RONDAS (5) iteraciones. En cada iteración llama a mostrarMemeYRealidades() (HU5) y comprobarRespuesta() (HU6).

Devuelve:int Puntuación final obtenida
static void mostrarPuntuacionFinal(int puntuacion)

HU8 — Muestra un recuadro con la puntuación final obtenida sobre el total de rondas.

puntuacionPuntuación total al terminar las 5 rondas
static void gestionarRanking(Scanner scanner, int puntuacion)

HU9 — Lee el ranking actual, comprueba si la puntuación entra entre las 3 mejores, y si es así pide el nombre al usuario y guarda la entrada en resultados/mejores.txt ordenada de mayor a menor.

Lanza:IOException si hay error al leer o escribir el fichero
static void mostrarRankingYDespedida()

HU10 — Lee y muestra las mejores puntuaciones guardadas en resultados/mejores.txt con formato numerado. Finaliza con un mensaje de despedida.

Lanza:IOException si hay error al leer el fichero
static void main(String[] args)

Punto de entrada del programa. Ejecuta el flujo completo: HU1 → HU2 → HU3 → HU4 → HU7 (HU5+HU6) → HU8 → HU9 → HU10.

argsArgumentos de línea de comandos (no se usan)
Lanza:Exception si ocurre algún error de E/S
