# DIRECTORIO de DOCUMENTACIÓN

En este directorio deben figurar:

1. Instalación. Un documento en formato Markdown, pdf o txt explicando:
	- Los requerimientos del programa (versión de JDK, etc.)
	- Cómo se debe proceder para descargar y compilar el programa.
	- Cómo se debe proceder para ejecutar el programa.
2. Manual de Usuario. Un documento en pdf que explique el propósito y funcionamiento del programa (incluyendo capturas de pantalla)
3. Manual del Programador. Un directorio con la documentación generada con javadoc.
